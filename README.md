\# AI-Powered Movie Recommendation System using PostgreSQL pgvector



\## Overview



This project demonstrates how PostgreSQL can be integrated with AI technologies using the pgvector extension.



\## Technologies



\- PostgreSQL

\- pgvector

\- Neon

\- DBeaver

\- Python

\- Pandas

\- Sentence Transformers



\## Dataset



MovieLens Small Dataset



\- 9,742 movies

\- 100,836 ratings



\## Features



\- Movie data storage

\- User rating analysis

\- Embedding generation using AI

\- Vector support with pgvector

\- Foundation for recommendation systems
